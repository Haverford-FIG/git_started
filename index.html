What is git started?

Git started is a designed to introduce members of the Haverford community to the fundamentals of software development. After the workshop, we'll have an opportunity for people to form groups to work on projects that they're passionate about.

What are we going to cover?

We're going to do a broad survey of things people should be learning in order to write software. We're going to be covering source control (git), front-end web techonlogies: html, css, JavaScript, backend technologies like SQL, and frameworks like Django. It's okay if you have no idea what any of these words mean. We're here to explain it to you.

Do I need any experience to attend?

Long story short: No. This lecture is designed for beginners and we really hope beginners will attend.
